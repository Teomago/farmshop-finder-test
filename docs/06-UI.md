# UI & Styling

## Stack
- TailwindCSS v4.
- HeroUI component library.
- Custom CSS modules / small utility CSS when needed.

## Patterns
- Layout & data fetch at server level; interactive widgets as client components.
- HeroUI for cards, buttons, images, modals.
